# Impeccable — System Index

## Overview

Impeccable is an open-source design skill for AI coding agents by Philip Bakaus. It provides a shared design vocabulary between the developer and AI agent — covering typography, color, motion, spatial design, interaction, responsive design, and UX writing — with explicit anti-pattern rules to prevent generic "AI slop" output.

**Source:** https://github.com/pbakaus/impeccable  
**Author:** Philip Bakaus  
**License:** Apache 2.0

## What's in this section

- **overview** — Impeccable's design philosophy, 7 domain reference areas, 23 commands, and the deterministic anti-pattern framework. Primary source for the "AI slop" problem framing and quality signal vocabulary used in the quality principles.
