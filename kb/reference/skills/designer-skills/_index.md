# designer-skills — System Index

## Overview

A collection of 8 agent skills for designers who prototype and build with AI coding tools, by Julian Oczkowski. The skills encode design process so AI follows a structured path instead of producing random output. Used via Claude Code, Cursor, Codex, and 40+ other AI coding agents.

**Source:** https://github.com/julianoczkowski/designer-skills  
**Author:** Julian Oczkowski  
**License:** Apache 2.0  

---

## What's in this section

The three skills used in Sistema's visual-language and quality principles synthesis:

- **grill-me** — Relentless interrogation skill that walks through every design decision until reaching shared understanding.
- **design-brief** — Turns a grilling session into a structured design brief. Includes codebase exploration, interview process, and a required output template.
- **frontend-design** — Builds with named aesthetic philosophies. Includes 8 philosophies (Dieter Rams, Swiss, Japanese Minimalism, Brutalist, Scandinavian, Art Deco, Neo-Memphis, Editorial) with concrete implementation parameters for each.
