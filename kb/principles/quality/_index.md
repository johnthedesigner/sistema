# Visual Quality — Principles

## Overview

Frameworks for evaluating whether design system decisions are specific and intentional versus generic AI defaults. This section provides the vocabulary and tests to identify low-quality token choices before they become embedded in the system.

---

## What's in this section

- **visual-quality-signals** — The AI slop test (first-order reflexes and second-order traps), absolute bans for token and component decisions, and the OKLCH color commitment level framework (Restrained → Committed → Full palette → Drenched). Use this before generating any token values.
