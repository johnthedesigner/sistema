# Spacing and Layout — Principles

## Overview

How to design a spacing system that functions as a unified coordinate system across component-internal, component-to-component, and layout-level spacing decisions.

---

## What's in this section

- **layout** — Base unit selection (4px vs. 8px), the scale as a unified system across three levels, density axis, breakpoint philosophy (content-first, minimize count), responsive token approaches, and the never-arbitrary rule.
